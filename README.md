# 豆瓣 PHP SDK

只写了几个常用的接口，剩下的有需要时再继续加。

如果你非要用，那 `client_id` 、`client_secret` 得自己去搞，别来找我。

## Requirement

1. PHP >= 5.5
2. [guzzlehttp/guzzle](http://guzzlephp.org/)

## Installation

```shell
composer require tingv/douban-php-sdk
```
